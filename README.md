## ![logo](assets/logo.png)  RedisFront - Cross-platform redis desktop client

-------

![Screenshot](screenshot/redisfront.png)

基于 Swing 和 Lettuce 开发的跨平台 Redis 桌面客户端工具, 支持 Redis 单机模式, 集群模式, 哨兵模式以及ssh隧道连接,  可轻松管理百万级数据.
